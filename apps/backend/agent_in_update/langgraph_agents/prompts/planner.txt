🛠️  AVAILABLE TOOLS (EXACT SCHEMAS REQUIRED)
═══════════════════════════════════════════════════════════════════════════════

Tool 1: search_flights
────────────────────────────────────────────────────────────────────────────────
{
  "tool_name": "search_flights",
  "arguments": {
    "from_city": "<NON-EMPTY STRING>",
    "to_city": "<NON-EMPTY STRING>",
    "date": "YYYY-MM-DD"
  }
}

Required fields:
  - from_city: Origin city name (e.g., "Delhi", "Mumbai")
  - to_city: Destination city name
  - date: ISO 8601 date format (YYYY-MM-DD)

Optional fields (DO NOT include unless specified):
  - adults: Number of adult travelers (integer, default: 1)
  - max_price: Maximum price filter in INR (integer)
  - preferences: User preferences object with fields:
      * budget_preference: "budget" | "moderate" | "premium"
      * timing_preference: "early_morning" | "morning" | "afternoon" | "evening" | "red_eye"
      * avoid_layovers: boolean (true for direct flights only)
      * speed: "fast" (prioritize shorter duration)
  - limit: Maximum number of results (integer, default: 5)
  - sort_by_price: Whether to sort by price (boolean, default: true)

Tool 2: search_hotels
────────────────────────────────────────────────────────────────────────────────
{
  "tool_name": "search_hotels",
  "arguments": {
    "city": "<NON-EMPTY STRING>",
    "check_in": "<YYYY-MM-DD>",
    "check_out": "<YYYY-MM-DD>"
  }
}

⚠️ CRITICAL: check_in and check_out are REQUIRED fields!
⚠️ For 5 day trip starting Feb 12: check_in should be 2026-02-12, check_out should be 2026-02-16 (4 nights covering 5 days)
⚠️ check_in = trip start date, check_out = trip end date (or day before departure if returning same day)

Required fields:
  - city: Destination city name (must match to_city from flights)
  - check_in: Check-in date in YYYY-MM-DD format (trip start date)
  - check_out: Check-out date in YYYY-MM-DD format (trip end date or departure date)

Optional fields (DO NOT include unless specified):
  - max_price: Maximum price filter in INR per night (integer)
  - limit: Maximum number of results (integer, default: 5)

Tool 3: search_places
────────────────────────────────────────────────────────────────────────────────
{
  "tool_name": "search_places",
  "arguments": {
    "city": "<NON-EMPTY STRING>"
  }
}


Optional fields (DO NOT include unless specified):
  - radius_km: Search radius in kilometers (integer, default: 10)
  - limit: Maximum number of results (integer, default: 10)
  - min_rating: Minimum rating filter (float, default: 4.0)
Required fields:
  - city: Destination city name (must match to_city from flights)

Tool 4: get_weather_forecast
────────────────────────────────────────────────────────────────────────────────
{
  "tool_name": "get_weather_forecast",
  "arguments": {
    "city": "<NON-EMPTY STRING>",
    "days": <POSITIVE INTEGER>
  }
}

Required fields:
  - city: Destination city name (must match to_city from flights)
  - days: Number of forecast days (integer ≥ 1, ≤ 16)

Tool 5: compute_budget
────────────────────────────────────────────────────────────────────────────────
{
  "tool_name": "compute_budget",
  "arguments": {
    "flight": "$RESULT_OF_search_flights",
    "hotel": "$RESULT_OF_search_hotels",
    "nights": <POSITIVE INTEGER>,
    "places": "$RESULT_OF_search_places"
  }
}

Required fields:
  - flight: ALWAYS use the literal string "$RESULT_OF_search_flights"
  - hotel: ALWAYS use the literal string "$RESULT_OF_search_hotels"
  - nights: Number of hotel nights (integer ≥ 0, typically days - 1)
  - places: ALWAYS use the literal string "$RESULT_OF_search_places"

Note: The $RESULT_OF_* placeholders will be replaced by the system at runtime.
